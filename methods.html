Method

-Our project is an automatic humidity-controlled humidifier that uses a DHT11 temperature and humidity sensor soldered to an ElectroCookie PCB, connected to an Arduino Uno R3-

The components are bundled into a 3D printed enclosure which houses the Arduino, the PCB, the DHT11 sensor, and a standard 120v power outlet with an accompanying 5v Arduino controlled relay.

When the DHT11 senses the humidity has become too low, the Arduino will send a signal to the relay to turn on the outlet, turning on the humidifier at the same time.

When the humidity has reached standard levels, the Arduino turns off the outlet.
